# Design Guide

포트폴리오용 UI 디자인 가이드

### Project Spec

- Vue `3.5.13`
- Vite `6.x`
- TypeScript `5.x`
- Ant Design Vue `4.x`

### Getting Started

```bash
npm install
npm run dev
```

### Scripts

```bash
npm run dev       # local server
npm run build     # production build
npm run preview   # preview build output
npm run typecheck # vue-tsc check
```
